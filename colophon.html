I chose to forgo implementing any framework and built this website from scratch using HTML (semantic markups and coming soon, microformats), CSS, and a few JS bits. My go-to code editor is
Visual Studio Code
. I intend to stick to HTML and CSS as much as possible. I will use JavaScript only when necessary. The source code for the site is hosted in a
public repository
 on GitHub. The whole project is deployed via
GitHub Pages
, as well as
Cloudflare
. At the moment, I am still figuring which deployment manager suits me best. As I work on this website, I have the following priorities in mind: 1) Accessibility 2) Mobile-first design principles 3) Security and privacy 4) Carbon footprint reduction 5) Speed and energy efficiency This site does not track its visitors nor does it allow web scrapers and crawlers. No cookies can be found either.
